# BeMod 🚀

Landing page desenvolvida para a **BeMod**, empresa focada no desenvolvimento de aplicações e soluções em TI.

🔗 Acesse: https://bemod.tech/

## 📌 Sobre o Projeto

Esta landing page foi criada com o objetivo de apresentar os serviços da empresa de forma moderna, profissional e dinâmica.

O foco principal do projeto foi:

- 💻 Design moderno e responsivo
- ✨ Uso de animações para melhorar a experiência do usuário

## 🛠 Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript  

## 🎨 Destaques

- Interface visual moderna  
- Animações complexas
